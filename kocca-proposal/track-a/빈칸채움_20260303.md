## 5. 상용화(비즈니스) 계획/전략

### ① 비즈니스 모델(상용화 방안)
본 과제의 상용화는 **B2C 구독 + B2B API + IP 라이선싱 연계**의 3축 수익모델로 추진한다.
- **B2C 구독형 SaaS**: 월 9,900원~29,900원 요금제(변환 건수/해상도/상업적 이용권 차등)
- **B2B API 라이선스**: 콘텐츠 제작사·마케팅 에이전시·문화기관 대상 월 구독형(월 100만~500만원)
- **IP 연계 수익**: 위즈데이터 보유 7080 만화작가 IP 기반 상업용 콘텐츠 제작 라이선스 제공
- **부가 수익**: 굿즈/프로필/전시용 이미지 패키지 제작 대행

또한 과제기간 내 실증으로 확보한 데이터를 기반으로, 2027년부터는 해외(일본·동남아) 한류 콘텐츠 수요처에 API/플랫폼 공급을 확대한다.

### ② 목표 소비층
- **B2C**: SNS 크리에이터, 유튜브 숏폼 제작자, 20~40대 일반 이용자, 레트로 콘텐츠 팬층
- **B2B**: 콘텐츠 제작사, 광고·브랜딩 에이전시, 출판·교육 기관, 전시/체험형 문화기관
- **선정 이유**: 이미지 콘텐츠 제작 빈도가 높고, 빠른 제작·차별화된 스타일 수요가 높은 집단으로 본 플랫폼의 효용(시간 단축, 스타일 일관성, IP 합법 활용)이 명확함

### ③ 목표 시장(국내/외) 및 진출 전략
- **국내(2026)**: 실증 중심 시장 진입
  - 3개 실증 트랙(SNS/크리에이터, 기업 마케팅, 전시/체험)으로 레퍼런스 확보
  - B2B 유료 PoC 계약 및 API 전환으로 초기 매출 창출
- **해외(2027~)**: 한류 IP 기반 차별화 진출
  - 일본·동남아권의 K-콘텐츠 팬덤 채널 대상 파트너십 추진
  - 다국어 UI/결제 및 해외 파트너 API 공급으로 확장

### ④ 홍보/마케팅 계획(월별)
- **4~5월**: 브랜드/랜딩페이지 정비, 베타 사전신청 모집
- **6~7월**: 크리에이터 협업 콘텐츠 제작, SNS 티징 캠페인
- **8월**: 클로즈드 베타 운영, 사용자 피드백 반영, 사례 인터뷰 확보
- **9월**: 오픈 베타 런칭, 성과형 광고(메타/유튜브) 집행
- **10월**: B2B 설명회/데모데이/IR 참가, 파트너 영업 집중
- **11월**: 실증 성과 홍보자료 발간(백서·영상), 후속 계약 전환
