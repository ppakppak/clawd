## [신청서] 2. 사업개요

### 과제명
**AI 화풍 변환 콘텐츠 플랫폼 "ToonStyle AI" 실증 제작**

### 사업 요약
한국 7080세대 대표 만화작가(이정문, 신문수 등)의 고유 화풍을 AI로 학습하여, 일반 사진을 해당 화풍으로 변환하는 콘텐츠 플랫폼을 개발하고 실증한다. Stable Diffusion XL 기반 LoRA 미세조정 + ControlNet 구조 가이드 + IP-Adapter-FaceID 얼굴 보존 기술을 자체 개발·통합하여, 작가별 화풍 특징을 정밀 재현하면서도 인물의 얼굴 정체성을 유지하는 AI 파이프라인을 구축하고, 이를 웹 기반 SaaS 플랫폼으로 서비스화하여 콘텐츠 제작 현장에서의 실증 결과를 도출한다.

### 핵심 키워드
생성형 AI, 이미지 스타일 변환, LoRA Fine-tuning, ControlNet, FaceID, 콘텐츠 IP 활용, 7080 만화 화풍

## [세부사업계획] 1. 과제 개요

### 1-1. 배경 및 필요성

**글로벌 AI 이미지 생성 시장의 폭발적 성장**
- 2024년 "지브리 풍 변환" 트렌드가 전 세계적으로 확산되며, AI 기반 이미지 스타일 변환에 대한 대중적 수요가 검증됨
- 글로벌 AI 이미지 생성 시장 규모: 2024년 약 15억 달러 → 2028년 약 80억 달러 전망 (CAGR 52%)
- 그러나 대부분의 서비스가 해외 IP(지브리, 디즈니 등)에 의존하며, **한국 고유 콘텐츠 IP를 활용한 서비스는 부재**

**한국 7080 만화 IP의 가치와 디지털 전환 필요성**
- 이정문(과학소년), 신문수(만화왕국) 등 7080세대 대표 만화작가들의 화풍은 높은 인지도와 향수를 보유
- 그러나 이들의 IP는 디지털화되지 못하고 정적 이미지 판매(건당 8만원)에 머물러 있음
- AI 기술을 통해 이러한 IP를 인터랙티브 콘텐츠로 확장하면, 한국 고유의 문화 자산을 글로벌 시장에 제공할 수 있음

**자체 기술 기반의 차별화**
- 범용 AI 서비스(ChatGPT, Midjourney 등)의 단순 활용이 아닌, **자체 개발한 txt2img + ControlNet + LoRA + FaceID 통합 파이프라인**을 통해 특정 작가의 화풍을 정밀 재현하면서 인물 얼굴 정체성 유지
- 이미 이정문 화백(51장), 신문수 화백(90장)에 대한 PoC를 완료하여 기술적 실현 가능성 검증
- **프로토타입 웹서비스 운영 중**: 사진 업로드 → 화백 선택 → 약 4초 내 변환 완료, 누적 68건 이상 변환 실적

**주관기관의 AI 기술 역량**
- 인튜웍스는 ETRI 투자로 설립된 AI 전문 기업으로, 최근 5년간 AI 바우처·데이터바우처 공급기업으로 다수 참여
- 산학연 Collabo R&D, 창업도약패키지, 디딤돌 창업성장 등 주관기관으로서 정부 R&D 수행 경험 풍부
- 상수도관로 AI 진단(iPipeCheck), 산업안전 AI 관제(RASS), 소 발정탐지 AI(CattleWatch) 등 다양한 도메인에서 컴퓨터비전 및 AI 솔루션 개발·사업화 이력 보유
- 특히 FaceNet 기반 얼굴 임베딩·유사도 매칭, dlib 68점 랜드마크 기반 안면 분석, YOLO 인스턴스 세그멘테이션 등 **이미지/얼굴 AI 기술의 실제 개발·적용 경험**을 보유하여, 화풍 변환 시 얼굴 특징 보존 및 이미지 처리 파이프라인 구축에 직접 활용 가능

- img2img 방식의 한계(원본 픽셀 잔류로 스타일 재현 저하)를 발견하고, **txt2img + ControlNet(canny) 방식으로 전환**하여 스타일 재현도 극대화
- LoRA를 UNet에 fuse한 뒤 FaceID 어텐션을 추가하는 **순서 의존적 통합 기법** 자체 개발
- 캡션 전략 연구: 이미지별 상세 캡션 vs 단순 캡션의 학습 효과 비교 → 최적 전략 도출 중

### 1-2. 과제 목표

**최종 목표:** 한국 7080 만화작가 화풍을 AI로 재현하는 콘텐츠 플랫폼 "ToonStyle AI"를 개발하고, 실제 콘텐츠 제작 현장에서 실증하여 시장 진입 기반을 확보한다.

**세부 목표:**
1. **AI 화풍 변환 엔진 고도화**: txt2img + ControlNet(canny) + LoRA(fuse) + IP-Adapter-FaceID 통합 파이프라인의 상용화 수준 안정화 및 추가 작가 확장
2. **웹 기반 SaaS 플랫폼 구축**: 사용자가 사진 업로드 → 작가/버전 선택 → 화풍 변환 결과물을 받는 웹 서비스 (프로토타입 이미 운영 중)
3. **작가 화풍 모델 확장**: 5인 이상의 7080 작가 화풍 모델 학습 완료 (현재 2인 완료)
4. **콘텐츠 제작 현장 실증**: 실제 콘텐츠 기업/크리에이터와 협업하여 3건 이상의 실증 사례 확보
5. **시장 진입**: 베타 서비스 오픈, 1,000명 이상 사용자 확보, IR 피칭 1회 이상

## [세부사업계획] 4. 인공지능 기술 활용 계획

### (4-1) 인공지능 기술이 플랫폼 및 콘텐츠 제작에서 수행하는 주요 기능

| AI 기술 | 기능 | 상세 |
|---------|------|------|
| **SDXL txt2img + LoRA** | 화풍 변환 핵심 엔진 | 작가별 화풍을 학습한 LoRA 가중치를 UNet에 fuse하여 텍스트→이미지 생성. img2img 대비 원본 픽셀 잔류 없이 순수 화풍 재현 |
| **ControlNet (Canny)** | 구조 가이드 | 원본 사진에서 Canny 엣지맵을 추출하여 구도·윤곽 유지. 사진 픽셀 직접 사용 없이 구조만 가이드로 전달 |
| **IP-Adapter-FaceID** | 얼굴 정체성 보존 | InsightFace(buffalo_l)로 512차원 얼굴 임베딩 추출 → IP-Adapter-FaceID로 생성 과정에 얼굴 특징 주입. 화풍 변환 후에도 인물 식별 가능 |
| **LoRA 버전 시스템** | 화풍 강도 제어 | 작가별 v1(기본)/v2(밸런스)/v2-strong(최대 화풍) 다중 버전 제공. 사용자가 목적에 맞게 선택 |
| **자동 캡셔닝** | 학습 데이터 전처리 | 이미지별 자동 분석(컬러/흑백, 배경, 구도) 기반 맞춤 캡션 생성. 트리거 워드 방식 |
| **품질 평가 AI** | 결과물 검증 | SSIM, LPIPS, FID 기반 자동 품질 스코어링 |
