## □ 사용 아키텍처(Architecture)

**사전학습 모델(SDXL) 기반의 LoRA 파인튜닝 + Img2Img 변환 아키텍처**

### 아키텍처 개요

```
[입력 사진] → [VAE Encoder] → [잠재 공간(Latent Space)]
                                        ↓
                              [U-Net + LoRA 가중치]  ←  [텍스트 프롬프트: "화백 스타일"]
                                        ↓
                              [VAE Decoder] → [출력: 만화풍 변환 이미지]
```

### 구성 요소

1. **Base Model: Stable Diffusion XL (SDXL) 1.0**
   - Stability AI에서 공개한 텍스트-이미지 생성 모델
   - U-Net 기반 노이즈 예측 네트워크 + VAE(Variational AutoEncoder) + 이중 텍스트 인코더(CLIP ViT-L + OpenCLIP ViT-bigG)
   - 1024×1024 해상도 지원

2. **LoRA (Low-Rank Adaptation) 모듈**
   - 화백별 화풍 특징을 학습하는 경량 어댑터
   - Network Dimension: 32, Alpha: 16
   - U-Net에만 적용 (722개 모듈)
   - 모델 크기: 약 163MB/화백 (전체 SDXL 6.9GB 대비 2.4%)

3. **Img2Img Pipeline (이미지-이미지 변환)**
   - 입력 사진에 노이즈를 추가한 후, LoRA가 적용된 U-Net으로 특정 화풍 방향으로 디노이징
   - Strength 파라미터(0.0~1.0)로 원본 유지도와 화풍 변환 강도를 제어
   - 최적 Strength: 0.60~0.65 (원본 인물 특징 유지 + 화풍 전환 균형)

### 동작 원리

1. **학습 단계**: 화백의 원화 이미지(50~100장)를 캡셔닝 후, SDXL 모델에 LoRA 어댑터를 부착하여 파인튜닝. 학습 시간은 RTX 4090 기준 약 30~40분.
2. **추론 단계**: 사용자가 업로드한 사진을 VAE로 잠재 공간에 인코딩한 후, 화백 스타일 프롬프트와 함께 LoRA가 적용된 U-Net으로 디노이징하여 만화풍 이미지를 생성.
3. **멀티 화백 지원**: 화백별 LoRA 가중치를 교체(swap)하는 방식으로 하나의 기본 모델에서 다수 화백의 화풍을 지원.

### B. AI 모델 테스트 결과서

#### 테스트 환경
- 테스트 사진: 2장 (커플 셀카, 개인 셀카)
- 변환 강도(Strength): 50%, 65%, 80% 3단계
- 추론 스텝: 30 steps, Euler-a 샘플러
- GPU: NVIDIA RTX 4090, 추론 시간: 2.7~4.1초/장

#### 정량 평가 지표 (신문수 화백 LoRA)

| Strength | SSIM ↑ | PSNR (dB) ↑ | Histogram Similarity ↑ |
|----------|--------|-------------|----------------------|
| 50% | 0.5800 | 20.65 | 0.8226 |
| **65%** | **0.4586** | **17.10** | **0.8162** |
| 80% | 0.3319 | 13.31 | 0.7645 |

- **SSIM (Structural Similarity Index)**: 원본 사진과 변환 이미지 간 구조적 유사도. 스타일 변환 특성상 0.3~0.6 범위가 적정 (과도하게 높으면 변환 미적용, 과도하게 낮으면 원본 특징 소실)
- **PSNR (Peak Signal-to-Noise Ratio)**: 원본 대비 픽셀 수준 차이. 스타일 변환에서는 참고 지표로 활용
- **Histogram Similarity**: 색상 분포 유사도. 화풍 변환 시에도 전체 색감 톤이 유지되는지 확인

## □ AI 솔루션 세부 설명

### 1. 솔루션 개요

**ToonStyle AI**는 사용자가 업로드한 사진을 한국 7080 만화작가의 고유 화풍으로 변환하는 생성형 AI 웹 서비스입니다.

최근 지브리 풍 AI 이미지 변환이 전 세계적으로 큰 트렌드를 형성한 가운데, ToonStyle AI는 한국 고유의 만화 IP를 활용하여 차별화된 서비스를 제공합니다.

### 2. 핵심 기술

| 기술 | 설명 |
|------|------|
| **LoRA Fine-tuning** | 소량 이미지(50~100장)로 특정 작가의 화풍을 학습하는 경량 파인튜닝 기법. 기존 대형 모델 전체를 재학습하지 않고 어댑터만 학습하여 비용과 시간을 크게 절감. |
| **Img2Img 스타일 변환** | 입력 사진의 구도와 인물 특징을 유지하면서 화풍만 변환하는 이미지-이미지 생성 기술. |
| **멀티 작가 LoRA 스왑** | 하나의 기본 모델에 작가별 LoRA 가중치를 동적으로 교체하여 다수 작가의 화풍을 지원. |

### 3. 서비스 구조

```
[사용자] → [웹 브라우저] → [FastAPI 서버]
                                ↓
                    [SDXL + LoRA 추론 엔진]
                    - GPU 서버 (RTX 4090)
                    - 화백별 LoRA 가중치 관리
                                ↓
                    [변환 결과 이미지 반환]
```

### 4. 활용 시나리오

1. **개인 사용자**: SNS 프로필 사진, 기념일 선물용 캐리커처
2. **기업 고객**: 브랜드 마케팅 콘텐츠, 캐릭터 굿즈 제작
3. **출판/미디어**: 도서 표지 일러스트, 뉴스 삽화 자동 생성
4. **교육**: 만화 미술 교육, 작가 화풍 분석 도구

### 5. 향후 고도화 계획

1. **얼굴 동일성 보존 강화**: ControlNet + IP-Adapter를 통합하여 변환 시 인물 얼굴 특징을 정확히 유지
2. **작가 풀 확대**: 현재 3인에서 10인 이상으로 지원 작가 확대
3. **실시간 변환**: 모델 최적화(TensorRT, ONNX)를 통한 1초 이내 추론 달성
4. **다양한 입력 지원**: 단일 사진뿐 아니라 영상(동영상) 프레임 변환
