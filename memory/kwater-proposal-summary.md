## 핵심 개발 항목

### 1. 데이터 수집/가공/오토라벨링
- K-Water 내시진단 영상 + 공동수급체 자체 영상 활용
- 유효 프레임 추출 (배경차분 알고리즘)
- 이미지 전처리 (노이즈/밝기/편향성/이상치 제거)
- **오토라벨링 솔루션 개발** (Instance Segmentation 기반)
  - 소량 휴먼라벨링 → 1차 모델 → 자동가공 → n차 반복 고도화
  - 성능: 정확도 95.3%, 속도 33fps (KOLAS 검증 완료)
- 데이터 포맷: json (좌표정보)

### 2. AI 모델 개발
- **Instance Segmentation**: Mask R-CNN, YOLACT, YOLO 등 비교 후 최적 선정
- 백본: ResNet, EfficientNet 등
- Loss: Focal Loss, IoU Loss
- 평가: mAP, IoU, F1-score
- 결함 클래스: 부식(결절), 도장재 박리, 슬라임, 균열, 침입수, 이물질 등

### 3. 크기/면적 산출 알고리즘
- **소실점(Vanishing Point)** 기반 원근 보정
- **Depth Estimation AI** (MiDaS, ZoeDepth 등)
- Volume Estimation (Point Cloud Projection)
- 단위구간(6m) 면적비 산출
- 식별면적 대비 결함면적비 계산

### 4. 관내부 영상 도식화(평면도)
- 어안렌즈 왜곡 보정 (체커보드 캘리브레이션)
- 카메라 자세 추정 (소실점 기반 회전량)
- **원통형 관로 → 평면 전개도(Strip) 변환**
- 실물크기(mm) 반영 평면도식 이미지 생성
- 히트맵/오버레이 시각화

### 5. 관상태 평가 알고리즘
- 평가점수: Si = Σ(Wi × Ai × Ci × Di)
  - Wi: 결함유형 가중치, Ai: 면적비, Ci: 위치보정, Di: 집중도
- 등급: A(건전) ~ D(교체권고)
- 환경부 상수도관망진단 매뉴얼(2021) 5등급 체계 준용
- 종합위험도: Ri = α×Si + β×ΔDi + γ×Bi

### 6. 레포트 자동생성
- PDF 레포트: 결함리스트, 상태지도, 등급요약, 권고사항
- Strip 전개도, 히트맵, 구간별 위험도 분포도 자동 삽입
- K-Water 기술보고서 양식 준용
- 웹 기반 레포트 뷰어 병행 개발

### 7. 시범적용 및 신뢰성 분석
- 공동수급체 테스트베드: ∅300~1100mm 다양한 관경
- 부단수 내시진단 현장 영상 수집
- 기술인 판독 결과와 AI 결과 비교
- 재학습 피드백 루프

## 우선 개발 필요 항목 (현재 → 1차년도 목표)
1. **오토라벨링 고도화**: 현재 수작업 polygon → AI 보조 세미오토 라벨링
2. **AI 모델 학습 파이프라인**: YOLO/Mask R-CNN 학습→평가→배포 자동화
3. **결함 크기 산출**: 소실점 기반 원근 보정 + 크기 추정
4. **데이터 확보**: 현재 721건 → 최소 3,000건 이상 목표
5. **wasvr GPU서버 연동 복구**: 학습 인프라 정상화
